<h1 align="center">WasmBoy</h1>
<h3 align="center">
  WasmBoy App for Open Web Desktop.
</h3>

## Overview

This app for Open Web Desktop is a Game Boy emulator based on torch2424/wasmboy.

## Installation

```bash
owd install-app @owdproject/app-wasmboy
```

## Usage

#### Available commands

```
wasmboy
wasmboy manager
```
